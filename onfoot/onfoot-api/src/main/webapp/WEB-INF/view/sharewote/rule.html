年终盘点活动规则

一、2017年终奖

1.查看年终成绩单

登录益拍APP，点击首页弹窗或焦点图进入2017年度盘点页面，查看2017年终成绩单，并可以领取20-25元随机金额年终奖。若您2017年没有上报记录，无法获得年终奖呦

2.参与投票奖励翻倍

活动期间将个人年终成绩单分享到微信好友、朋友圈，邀请微信好友为您投票。活动结束后根据票数的排名，获得不同的奖励奖励翻倍，最高可达15倍

每人每天只有1票，当天未投票，票数清零;第2天可再次为好友投票;投票排行榜每2分钟刷新1次。

3.活动时间

年终奖领取：2月16日 0点-2月23日 0点;

投票时间：2月16日 0点-2月23日 0点;

年终奖发放：2月24日

二、新用户注册得红包

在2月16日0点-2月23日 0点活动期间内新注册益拍的用户，即可领取16元新年红包，登录账号后在“我的-收入”中查看
